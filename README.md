📌 Sobre o Projeto
Este é um quiz interativo desenvolvido utilizando apenas HTML e CSS. O objetivo é proporcionar um ambiente simples e visualmente atrativo para testar conhecimentos em programação, sem a necessidade de JavaScript.

🚀 Funcionalidades
✅ Perguntas de múltipla escolha estilizadas com CSS puro.
✅ Layout responsivo para diferentes dispositivos.
✅ Design moderno e intuitivo.

🛠 Tecnologias Utilizadas
HTML5 para a estrutura do quiz.
CSS3 para estilização e interatividade visual.

🎯 Objetivo do Projeto
Este projeto foi criado para demonstrar como é possível construir um quiz estiloso e funcional apenas com HTML e CSS, sem a necessidade de JavaScript.
